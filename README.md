# Learn HTTP Servers in Typescript

Boot.dev project to learn the basics of HTTP servers. 
[Project Guide](https://www.boot.dev/courses/learn-http-servers-typescript)

## Servers + Routing
Used the [Express web framework](https://expressjs.com/) to handle resources and middleware.

## Databases
Used [Postgres](https://www.postgresql.org/) + [Drizzle](https://orm.drizzle.team/) to generate + migrate a database for persistent information.

## Authentication + Authorization
Used Javascript Web Tokens (JWTs) and Refresh Tokens to manage user authentication and authorization

## Webhooks
Used basic webhooks to support 3rd-party inputs and maintain idempotence.
